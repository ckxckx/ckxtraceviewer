

# ckxtraceviewer



### Install:

```
git clone https://github.com/ckxckx/ckxtraceviewer.git
cd ckxtraceviewer 
pip install .
cp ./ckxtraceviewer_plugin.py <YOUR IDA PLUGINS DIR or idaapi.get_user_idadir()/plugins/>
```
